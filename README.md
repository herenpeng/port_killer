# 端口进程杀手 (Port Killer)

一个基于Python和Tkinter的图形化工具，用于查找并杀死占用指定端口的进程。

## 功能特性

- 🎯 通过端口号查找占用进程
- 📊 显示详细的进程信息（协议、地址、状态、PID）
- 🔪 一键杀死选中的进程
- 🔄 实时刷新功能
- 💻 简洁友好的图形界面
- ⚡ 支持双击快速杀死进程
- 📦 支持打包为独立exe文件

## 系统要求

- Windows 操作系统
- Python 3.6+
- tkinter（通常随Python一起安装）

## 使用方法

### 方式一：直接运行Python脚本

```bash
python port_killer.py
```

### 方式二：使用打包后的exe文件

1. 运行打包脚本生成exe：
   ```bash
   build.bat
   ```

2. 在 `dist` 文件夹中找到 `PortKiller.exe`

3. 双击运行即可使用

## 打包说明

### Windows系统

**自动打包（推荐）：**

直接运行批处理脚本：
```bash
build.bat
```

该脚本会自动：
- 检查Python环境
- 安装PyInstaller（如果未安装）
- 执行打包流程
- 打开生成的exe文件所在目录

**手动打包：**

```bash
# 1. 安装PyInstaller
pip install pyinstaller

# 2. 执行打包
pyinstaller --clean build.spec

# 3. 查看生成的文件
dist\PortKiller.exe
```


## 打包选项说明

`build.spec` 配置文件说明：

- `console=False`: 不显示控制台窗口（GUI模式）
- `upx=True`: 使用UPX压缩减小程序体积
- `icon='icon.ico'`: 设置程序图标（可选）
- `name='PortKiller'`: 生成的可执行文件名称

## 自定义打包

### 添加程序图标

将 `icon.ico` 文件放置到项目根目录，重新打包即可。

### 调试模式

如需查看错误信息，可以临时启用控制台：

修改 `build.spec` 中：
```python
console=True,  # 显示控制台窗口
```

## 界面说明

- **端口号输入框**：输入要查询的端口号
- **查找进程按钮**：执行端口查询
- **杀死选中进程按钮**：结束选中的进程
- **刷新按钮**：重新查询当前端口的进程状态
- **进程列表**：显示占用端口的进程详情
  - 协议：TCP/UDP
  - 本地地址：本机IP和端口
  - 外部地址：远程IP和端口
  - 状态：连接状态
  - PID：进程ID

## 注意事项

⚠️ **重要提示：**

1. 某些系统进程可能需要管理员权限才能结束
2. 结束进程可能导致正在运行的程序异常退出，请谨慎操作
3. 建议在结束进程前确认该进程不会影响系统或其他重要应用
4. 首次运行时，Windows Defender可能会提示未知程序，请选择允许运行

## 常见问题

### Q: 提示"拒绝访问"？
A: 尝试以管理员身份运行程序

### Q: 找不到占用端口的进程？
A: 确认端口号输入正确，或者该端口确实被占用

### Q: exe文件太大？
A: 这是正常现象，包含了Python运行时库。可以使用单文件打包模式减小总体积。

### Q: 如何查看当前占用的端口？
A: 可以打开命令行输入 `netstat -ano` 查看所有端口占用情况

### Q: 打包后杀毒软件报毒？
A: PyInstaller打包的程序可能被误报，可以：
- 添加到杀毒软件白名单
- 使用代码签名证书
- 向杀毒软件厂商提交误报
